**# Stock Management System**

Inventory management system developed for study and portfolio purposes, focused on simulating real business operations and stock control routines.

## Technologies

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Features

- User authentication
- Product management
- Stock movements
- Reverse logistics
- Inventory history
- Dashboard and reports

## Database Concepts

- Relational modeling
- Foreign Keys
- Constraints
- Views
- Triggers

## Project Status

Currently under development.

Implemented so far:
- Database structure
- Relational modeling
- Test data population
- SQL training queries
- Initial project organization

## Future Improvements
